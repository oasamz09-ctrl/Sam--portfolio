Requirements Specification

MySanity AI Behavioural Intelligence Platform

Structured Business and System Requirements for an AI-powered mental health platform designed to enable proactive wellbeing through mood tracking, sleep analysis, journaling, and conversational AI (text + voice).

Business Requirements

  • BR1: Provide a unified mental health platform combining mood, sleep, journaling, and AI interaction.
  • BR2: Enable proactive mental health awareness through continuous behavioural tracking and insights.
  • BR3: Improve user engagement by delivering personalised, context-aware wellbeing feedback.
  • BR4: Reduce fragmentation of mental health tools into a single integrated ecosystem.
  • BR5: Support early and proactive identification of behavioural patterns linked to emotional wellbeing changes.

Functional Requirements

  • FR1: Users must be able to log daily mood states and emotional wellbeing entries.
  • FR2: System must capture and analyse sleep duration and sleep quality data.
  • FR3: Users must be able to create and store journal entries for emotional reflection.
  • FR4: System must support AI-powered conversational interaction via text input.
  • FR5: System must support AI conversational interaction via voice input and output.
  • FR6: System must generate personalised insights based on combined behavioural datasets.
  • FR7: System must provide a dashboard showing trends across mood, sleep, and behaviour.
  • FR8: System must store historical user data to enable long-term behavioural analysis.
  • FR9: System must be able to recognise patterns and user trends
  • FR10: System must have Mindfulness and Coping exercises and be able to suggest based on historical user input

Non-Functional Requirements

  • NFR1: AI responses must be generated within an acceptable low-latency threshold (≤ 2–3 seconds).
  • NFR2: All user data must be encrypted and securely stored due to sensitivity of mental health data.
  • NFR3: System must be scalable to support increasing behavioural data volume over time.
  • NFR4: Platform must maintain high availability to ensure continuous user access.
  • NFR5: Voice interaction must support reliable real-time processing with minimal delay.
  • NFR6: System must be designed with privacy-first principles and GDPR compliance in mind.

User Stories

    Authentication

    • As a new user, I want to create an account so that my wellbeing data is securely stored.
    • As a user, I want to log in securely so that only I can access my wellbeing information.
    • As a user, I want to reset my password so that I can regain access if I forget it.

    Mood Tracking

    • As a user, I want to log my mood daily so that I can understand my emotional patterns.
    • As a user, I want to view my mood history so that I can identify changes in my emotional wellbeing over time.

    Sleep Tracking

    • As a user, I want to track my sleep so that I can identify how rest affects my mental health.
    • As a user, I want to view my sleep trends so that I can recognise healthy or unhealthy sleeping habits.

    AI Assistant

    • As a user, I want to chat with an AI assistant so that I can receive personalised wellbeing support.
    • As a user, I want to communicate with the AI using my voice so that I can interact naturally.
    • As a user, I want the AI to personalise recommendations based on my mood, sleep, completed exercises and previous insights so that the advice is relevant to my current wellbeing.
    • As a user, I want the AI to explain why it has made a recommendation so that I understand how it relates to my wellbeing data.

    Mindfulness & Wellbeing

    • As a user, I want to access mindfulness and coping exercises so that I can improve my mental wellbeing.
    • As a user, I want to mark exercises as completed so that I can track my progress.
    • As a user, I want suggestions on how to improve my mental health instead of only viewing insights.

    Insights & Progress

    • As a user, I want to receive personalised insights based on my wellbeing data so that I can better understand my behavioural patterns.
    • As a user, I want to receive weekly wellbeing summaries so that I can reflect on my progress.
    • As a user, I want to view trends across my mood, sleep and completed activities so that I can make informed lifestyle improvements.

    Community

    • As a user, I want to join communities of like-minded people so that I can receive encouragement and motivation from others.

    Privacy & Security

    • As a user, I want my wellbeing data to be securely stored so that my personal information remains private.
    • As a user, I want to control what information is used by the AI so that I have confidence in how my data is handled.
