MySanity AI Behavioural Intelligence Platform

Mapping how users interact with the system across mood tracking, sleep logging, and AI conversational engagement (text + voice).

End-to-End User Journey

The MySanity platform is designed around continuous behavioural input and feedback loops. Each user action contributes to a growing intelligence model.

1. Onboarding Journey

  • User signs up and creates a wellbeing profile
  • Initial mood baseline is captured
  • Sleep preferences and patterns are logged
  • System introduces AI assistant capabilities

2. Mood Logging Flow

  • User selects daily mood input (scale or emotion selection)
  • Optional context is added (text or voice)
  • Data is stored in behavioural dataset
  • AI identifies short-term emotional trends

3. Sleep Tracking Flow

  • User logs sleep duration and quality
  • System aggregates historical sleep data
  • AI detects patterns between sleep and mood
  • Insights are generated for wellbeing improvement

4. AI Interaction Flow (Text & Voice)

  • User interacts with AI via text or voice input
  • System processes intent and emotional context
  • AI responds with personalised guidance
  • Conversation history contributes to behavioural model

System Feedback Loop

All user interactions feed into a continuous loop: Data Input → Behavioural Analysis → AI Insight → User Feedback
