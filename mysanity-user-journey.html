User Journey

MySanity AI Behavioural Intelligence Platform

Mapping how users interact with a proactive mental health system combining mood tracking, sleep analysis, journaling, and AI conversational support (text + voice).

Journey Overview

The MySanity user journey is built around continuous behavioural feedback loops. Each interaction contributes to a deeper understanding of the user’s mental wellbeing and supports proactive intervention rather than reactive support.

1. Onboarding & Baseline Setup

  • User creates a personal wellbeing profile
  • Initial mood and emotional baseline is captured
  • Sleep habits and behavioural patterns are recorded
  • System introduces AI assistant capabilities (text + voice)
  • Consent and privacy preferences are established (mental health data sensitivity)

2. Mood Tracking Journey

  • User logs daily mood using emotion scale or descriptive input
  • Optional context is captured via text or voice input
  • Data is stored and mapped against behavioural history
  • AI identifies short-term emotional fluctuations and triggers
  • Insights are generated to support proactive awareness

3. Sleep & Recovery Tracking

  • User logs sleep duration and perceived sleep quality
  • System aggregates historical sleep behaviour
  • AI identifies correlation between sleep patterns and mood changes
  • Insights highlight lifestyle impact on mental wellbeing
  • Recommendations support improved recovery habits

4. AI Interaction Flow (Text & Voice)

  • User interacts with AI assistant via chat or voice input
  • System interprets emotional tone and intent
  • AI responds with contextual, personalised wellbeing guidance
  • Conversation history contributes to behavioural intelligence model
  • AI adapts responses based on long-term user patterns

System Feedback Loop

MySanity operates on a continuous intelligence loop:

Data Input → Behavioural Analysis → AI Insight Generation → User Reflection → Improved Behavioural Outcomes
